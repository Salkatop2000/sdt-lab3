# sdt-lab3 README

Это плагин для генерации блоков if, else, for и while

## Функционал

### ifCommand

Генерирует блок if

### elseCommand

Генерирует блок else

### if-elseCommand

Генерирует блок if-else

### forCommand

Генерирует блок for

### whileCommand

Генерирует блок while

## Подробные описания функций

## activate

- Принимает на вход контект в котором работает плагин
- Ничего не возвращает
- регистрирует команды плагина

## registerCommand

- принимает на вход внутренне название команды и функцию, вызываемую по этой команде
- возвращает объект команды
- регистрирует команду для дальнейшего использования

## log

- принимает текст сообщения
- ничего не возвращает
- выводит вообщение в консоль

## push

- приниматет на вход команды
- ничего не возвращает
- добавляет команды в массив подписок
